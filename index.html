Capabilities

Everything your pipelines need

From deterministic pattern matching to LLM-powered root cause analysis — with a 5-gate safety system before any remediation is touched.

Hybrid Diagnosis
7 deterministic regex matchers run first. LLM only invoked for novel failures — reducing latency and cost by 74% on average.
Real-time Watchdog
Polls orchestrators every 5 minutes. Redis debounce prevents duplicate triage. Instantly escalates critical failures.
5-Gate Remediation
confirm + approver_id + plan exists + auto_remediation_enabled + risk level. All five gates must pass. No silent changes.
MCP Native
10 MCP tools — plug directly into Claude Desktop. list_pipelines, diagnose_failure, propose_remediation, file_incident and more.
OpenLineage Tracing
Traverse upstream/downstream dependency graphs via Marquez. Understand blast radius before proposing any fix.
Full Observability
Prometheus metrics, structured logs via structlog, OpenTelemetry traces, and a pre-built Grafana dashboard included.

Workflow

How DataGuard works

1
Watchdog polls
Checks Airflow & Argo every 5 min. Redis debounce prevents duplicate triage runs.
2
Pattern match
7 regex matchers for OOM, timeouts, schema drift, key errors. LLM skipped if match found.
3
TriageAgent runs
Agentic loop: get logs → trace lineage → diagnose → propose remediation → file incident.
4
Human approves
5-gate safety check. execute_remediation only runs after explicit human sign-off.

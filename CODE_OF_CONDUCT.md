# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

**TL;DR:** Be direct and technical. Critique code and ideas, not people. No harassment, discrimination, or personal attacks. Maintainers will remove contributors who cannot meet this bar.

Report violations to `conduct@dataguard-agent.io`. All reports are reviewed within 72 hours and kept confidential.
